Artist Journey and Comparison Framework

This is built on synthetic data.

An interactive dashboard over a synthetic artist roster and a pool of a few hundred artists. The development quadrant places every artist by total yearly placements versus total yearly streams on a fixed plane normalized against the pool, and continues each path with a recursive forecast. Other tabs trace a single artist over time, align artists on career age with both raw and normalized values, and rank the roster for investment. Every number comes from one seeded simulation.

Quadrant options

History off shows only each artist's latest position. Predictions continue each path with the recursive forecast, drawn with dashed markers at the 1, 2, 5, and 10 year ahead horizons.
